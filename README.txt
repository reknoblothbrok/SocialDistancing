Este es un programa de visión computacional capaz de detectar personas, si hay más de una persona dentro de la imagen/video/video en tiempo real, calcula la distancia que hay entre ellas y las encierra en un recuadro color verde si es una distancia segura y en un recuadro color rojo si no lo es.

Verisón de python a instalar
python3.8.5
librerías a instalar
pip install opencv-contrib-python==4.4.0.42
pip install imutils==0.5.3
pip install numpy==1.18.1

Descargar los modelos preentrenados YOLOV3 de este enlace:
https://pjreddie.com/darknet/yolo/
